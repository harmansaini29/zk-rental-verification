# 🔐 ZK Rental Verification System (Karamstay)

## 🚀 Overview

A privacy-first tenant verification system using Zero-Knowledge Proofs (ZKPs) to eliminate fraud and protect user data in rental platforms.

This project is part of an ideathon submission for zkVerify.

---

## ❗ Problem

Rental platforms (PGs, hostels, flats) face:

* Fake tenant identities
* Screenshot-based proof (easy to manipulate)
* Privacy risks due to sharing sensitive documents

---

## 💡 Solution

A ZK-powered verification layer where tenants can prove:

* Identity (KYC verified)
* Payment reliability
* Rental history

WITHOUT revealing raw personal data.

---

## 🧠 Key Features

* Zero-Knowledge Tenant Verification
* Verifiable Rental Reputation Score
* Fraud-proof owner dashboard
* Privacy-preserving onboarding

---

## ⚙️ Architecture (Planned)

* Frontend: Flutter App
* Backend: Node.js + MongoDB
* zk Layer: zkVerify (Proof verification layer)

System stores only:
→ Proofs
→ Verification status
NOT raw user data

---

## 🔗 zkVerify Integration (Planned)

* Verify SNARK/STARK proofs
* zkTLS-based external data validation
* Future: Verifiable AI scoring
